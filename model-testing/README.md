# Model Testing (Lightweight)

A **lightweight template** to document model behavior, test conditions, and evaluation results without exposing proprietary code.

## Suggested Uses
- Compare prompts or temperature settings and record differences.
- Document fallback behavior and error handling.
- Log examples of great vs. problematic outputs for training discussions.
